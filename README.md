\#게시판 프로젝트

1. 기능

* CRUD란? 데이터를 다루는 기본 4가지 기능. 이 기준으로 기능을 생각하자.
* C (Create) : 최초 게시판 생성(틀), 글 작성, 아이디 생성
* R (Read) : 게시판 목록 읽기, 글 수정 시 기존 내용 불러오기, 아이디 확인, 글 클릭시 해당 게시물 확인
* U (Update) : 해당 단축키를 누르면 게시판 목록 활성화, 글 수정, 비밀번호 수정, 로그아웃
* D (Delete) : 글 삭제, 아이디 삭제

2. 기술 스택

* 프레임워크 : Flask
